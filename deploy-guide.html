网站更新操作流程

Step 1
打开 Terminal，进入项目目录
cd YUE-homesite
Step 2
切换到主分支
git checkout main
Step 3
添加或修改文件夹里的内容
在项目文件夹中进行你需要的修改，例如更新图片、文字或页面内容。
Step 4
提交代码改动
git add . git commit -m "你的改动描述" git push origin main
Step 5
部署到 GitHub Pages
npm run deploy
npm run deploy 会自动执行以下步骤：
  • npm run build — 编译源码，生成 dist/
  • gh-pages -d dist — 把 dist/ 推送到 gh-pages 分支
部署完成后稍等 1-2 分钟，网站就会更新。
